# Solar-System
An immersive, educational 3D exploration of our solar system built with Three.js. This project features smooth orbital mechanics, interactive planetary data, and a responsive camera system that allows users to fly through space and learn about our cosmic neighbors.

🚀 Features

Interactive Exploration: Click or tap on any planet to zoom in and view detailed information.

Dynamic Orbits: Planets orbit the Sun at relative speeds, creating a living model of the solar system.

Advanced Camera Controls: Includes OrbitControls for zooming, panning, and rotating the view.

Synthwave Aesthetics: A beautiful starfield background and neon-inspired planetary glow.

Responsive Design: Works seamlessly on desktops, tablets, and smartphones.

🛠️ Built With

HTML5/CSS3 - Layout and UI panels.

Three.js - 3D rendering engine.

OrbitControls.js - Interaction and navigation.

🕹️ How to Play / Use

Navigate: Click and hold to rotate the camera around the Sun.

Zoom: Use your mouse wheel or pinch-to-zoom on mobile.

Inspect: Click on a planet to fly toward it and open the information panel.

Return: Click the "Back to Orbit" button to return to the system overview.

📖 Planets Included

The Sun: Our central star.

Terrestrial Planets: Mercury, Venus, Earth, Mars.

The Giants: Jupiter, Saturn (with rings), Uranus, and Neptune.

Created for educational purposes to demonstrate 3D web graphics and interactive storytelling.
